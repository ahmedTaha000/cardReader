
# Project Wiki

## Overview

This project is an Android application for reading, writing, and formatting NFC cards. It leverages Kotlin and Jetpack Compose for the user interface and interacts with NFC hardware for its core functionality.

## Key Features

*   **Read NFC Cards:** Read data from NFC cards.
*   **Write to NFC Cards:** Write data to NFC cards.
*   **Format NFC Cards:** Format NFC cards.
*   **AES Encryption:** Encrypt and decrypt data using AES.

## Getting Started

### Prerequisites

*   Android Studio
*   An Android device with NFC support

### Installation

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Build and run the application on your Android device.

## Usage

1.  **Main Activity:** The main screen of the application provides options to read, write, or format an NFC card.
2.  **Read Activity:** The read screen displays the data read from an NFC card.
3.  **Write Activity:** The write screen allows you to input data to be written to an NFC card.
4.  **Format Activity:** The format screen formats the NFC card.
